## Getting Started

To get a local copy up and running, follow these steps:

### Prerequisites

Make sure you have Node.js installed (v18.0.0 or higher recommended).

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/randallmurphy/randall-murphy-portfolio.git
   ```
2. Install NPM packages
   ```bash
   npm install
   ```
3. Run the development server
   ```bash
   npm run dev
   ```
4. Build for production (Optional)
   ```bash
   npm run build
   ```

## Design Language: Liquid Brutalism

The aesthetic merges raw, brutalist structural elements (heavy typography, distinct borders, dark backgrounds) with fluid, "liquid" animations and spatial components leveraging `framer-motion` and `three.js`. The result is a clean, hyper-modern, and dynamic spatial experience.
